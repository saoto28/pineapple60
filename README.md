# pineapple60
ergonomic keyboard with Trackpoint

The purposes of this project.
- Install inside ThinkPad T470s (compatible with T460s)
- Install inside the Trackpoint keyboard (KU-1255 ~~and KT-1255~~)

(They are alomost same size.)
